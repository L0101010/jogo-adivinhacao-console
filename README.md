# Jogo de Adivinhação (Console - C#)
Um jogo simples feito em C# onde o jogador precisa adivinhar um número aleatório gerado pelo sistema.

## Como funciona
O programa gera um número aleatório entre 1 e 20, e o jogador deve tentar adivinhar qual é esse número.

A cada tentativa, o sistema informa se o número digitado é maior ou menor que o número secreto.

O jogo continua até o jogador acertar.

## Funcionalidades
* Geração de número aleatório
* Sistema de tentativas
* Dicas (maior ou menor)
* Contador de tentativas
* Interação via console

## Tecnologias utilizadas
* C#
* .NET (Console Application)

## Como executar
1. Clone o repositório:
```
git clone https://github.com/seu-usuario/jogo-adivinhacao-console.git
```
2. Abra o projeto no Visual Studio
3. Execute o programa

## Exemplo de uso
```
Digite um número:
10
O número é maior que isso!

Digite um número:
15
Parabéns! Você acertou!
```
